# Parkway East Design - Blog Section

## Structure

Each blog article is in its own folder containing:
- `content.md` - The full article text
- `images/` - All images used in that article

```
blog/
├── README.md                           ← This file
├── 01-finding-your-design-style/
│   ├── content.md                      ← Article copy
│   └── images/                         ← 22 images
│       ├── 6-scaled.webp                (hero image)
│       ├── 14-1.webp through 34.webp     (style examples)
│       └── ...
└── 02-the-heart-of-st-george/
    ├── content.md                      ← Article copy
    └── images/                         ← 27 images
        ├── Blog-Covers-1.webp           (hero image)
        ├── 8.webp through 98.webp        (location photos)
        └── ...
```
